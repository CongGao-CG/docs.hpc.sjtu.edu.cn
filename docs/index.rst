Pi超算集群用户文档
==================

`docs <https://docs.hpc.sjtu.edu.cn/>`_ 是上海交通大学 Pi2.0 超级计算集群用户文档，为用户提供快速上手指导和问题解答。如果想获取更多 hpc 团队的动态和信息，请访问 `hpc <https://hpc.sjtu.edu.cn/>`_ 。   

.. caution:: *超算平台禁止**运行军工项目等涉密计算任务。*

.. tip:: 该文档通过 `github仓库 <https://github.com/SJTU-HPC/docs.hpc.sjtu.edu.cn>`__\ 维护。如果您有什么建议或者意见，欢迎提交修改。
