帐号说明
==========

在交我算的帐号体系中，帐号会被分为主帐号和子帐号。

主帐号是收缴费主体，由课题组负责人申请。

主帐号与子帐号均为独立帐号，仅在计费关系上存在关联。

每个主帐号能免费申请子帐号，个数不限，请按需申请。


主帐号和子帐号申请
====================
由帐号负责人或帐号使用人在“交我办”（或 `我的数字交大 my.sjtu.edu.cn <https://my.sjtu.edu.cn>`_ ）中的“交我算”里申请。我们将会在两个工作日内开通帐号。
