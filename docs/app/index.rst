PI上的软件模块
--------------

Pi有许多预建的软件模块，并且数量还在不断增长。欢迎您告诉我们您研究领域中流行的软件。由于收费很少甚至为零，因此开源软件的安装优先级更高。

Pi上的软件可以分类为编译器和平台，MPI库，Math库，FD工具，生物信息学工具等。
